DATABASE DELTREE

Synopsis

Removes database keytree/value

Description

Deletes a family or specific keytree within a family in the Asterisk database.

Returns 1 if successful, 0 otherwise.

Syntax

DATABASE DELTREE FAMILY KEYTREE
Arguments
  • family
  • keytree
