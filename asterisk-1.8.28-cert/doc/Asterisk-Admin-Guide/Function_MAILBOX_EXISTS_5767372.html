MAILBOX_EXISTS()

Synopsis

Tell if a mailbox is configured.

Description

Returns a boolean of whether the corresponding mailbox exists. If context is not specified, defaults to the default context.

Syntax

MAILBOX_EXISTS(mailbox@context)
Arguments
  • mailbox
  • context
