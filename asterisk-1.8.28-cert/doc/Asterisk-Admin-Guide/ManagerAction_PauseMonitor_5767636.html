PauseMonitor

Synopsis

Pause monitoring of a channel.

Description

This action may be used to temporarily stop the recording of a channel.

Syntax

Action: PauseMonitor
ActionID: <value>
Channel: <value>
Arguments
  • ActionID - ActionID for this transaction. Will be returned.
  • Channel - Used to specify the channel to record.
